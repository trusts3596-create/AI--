# TravelBot 移动端优化指南

## 📱 移动端特性

TravelBot 智慧旅游规划助手已经针对移动设备进行了全面优化，提供更好的用户体验：

### 🎯 核心优化特性

#### 1. 响应式设计
- **自适应布局**：完美适配手机、平板等不同屏幕尺寸
- **触摸友好**：所有交互元素都经过触摸优化，符合 Material Design 标准
- **手势支持**：支持触摸、滑动等手势操作

#### 2. 移动端侧边栏
- **滑入式设计**：侧边栏从左侧滑入，不占用聊天空间
- **覆盖层**：半透明背景遮罩，便于专注于当前对话
- **一键关闭**：点击遮罩或关闭按钮快速关闭侧边栏

#### 3. 智能键盘处理
- **自动调整**：键盘弹出时自动调整界面布局
- **防止遮挡**：输入框始终保持在可视区域内
- **自动聚焦**：页面加载时自动聚焦输入框

#### 4. 移动端导航栏
- **底部导航**：固定在底部的导航栏，便于单手操作
- **图标导航**：直观的图标设计，无需文字说明
- **快速切换**：一键切换不同功能模块

#### 5. 触摸优化
- **反馈效果**：点击时有视觉反馈，提升交互体验
- **触摸目标**：所有按钮都符合 44px 最小触摸目标标准
- **防误触**：合理的间距设计，避免误操作

## 🚀 使用方法

### 1. 移动端访问

#### WiFi 访问
1. 确保手机和电脑连接到同一 WiFi 网络
2. 启动 TravelBot 服务器
3. 在手机浏览器中输入电脑的局域网 IP 地址：
   - 例如：`http://192.168.43.8:5000`

#### 局域网发现
1. 服务器启动时会显示可访问的 URL
2. 使用显示的局域网地址进行访问

### 2. 推荐浏览器

- **Chrome**：推荐使用 Chrome 移动浏览器
- **Safari**：iOS 设备上的 Safari 浏览器
- **Firefox**：Firefox 移动浏览器

### 3. 使用技巧

#### 侧边栏操作
- 点击左上角菜单按钮打开侧边栏
- 点击胶囊按钮快速发送预设问题
- 点击关闭按钮或遮罩层关闭侧边栏

#### 聊天操作
- 点击输入框自动聚焦
- 使用 `Enter` 发送消息，`Shift+Enter` 换行
- 点击发送按钮或按 `Enter` 发送消息

#### 导航操作
- 点击底部导航栏图标切换功能
- 聊天模式：当前页面
- 历史记录：查看对话历史（开发中）
- 设置：个性化设置（开发中）
